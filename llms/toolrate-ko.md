# ToolRate 시스템 개요

## ToolRate란?

ToolRate는 자율 AI 에이전트를 위한 **크라우드소싱 기반 신뢰성 레이어**입니다 — 에이전트가 외부 도구나 API를 호출하기 *전에* 신뢰도를 평가할 수 있도록 해주는 실시간 신뢰성 오라클입니다.

이는 에이전트 개발에서 가장 중요한 실질적 문제 중 하나를 해결합니다: 대부분의 장애는 LLM 자체가 아니라 외부 도구 및 API의 예측 불가능한 동작 — 속도 제한, 스키마 드리프트, 인증 문제, 봇 방지 보호, 엣지 케이스 — 에서 비롯됩니다.

---

## ToolRate는 누구를 위한 것인가요?

- **프로덕션 수준**의 AI 에이전트를 구축하는 개발자
- **LangChain, CrewAI, LangGraph, AutoGen** 또는 **LlamaIndex**로 작업하는 팀 및 개인 개발자
- **GDPR 및 데이터 레지던시**를 중요시하는 유럽 개발자
- 데모에서는 잘 작동하지만 실제 환경에서 자주 실패하는 에이전트에 불만을 느끼는 모든 분

---

## ToolRate의 작동 방식

시스템은 의도적으로 단순하고 가볍게 설계되었습니다:

**1. 호출 전 점검**

외부 도구나 API를 호출하기 전에 에이전트가 ToolRate에 쿼리를 보냅니다:

```python
assessment = toolrate.guard(tool_identifier=..., context=...)
```

**2. 구조화된 응답**

ToolRate는 다음 내용을 담은 JSON 페이로드를 즉시 반환합니다:

| 필드 | 설명 |
|---|---|
| `reliability_score` | 0~100점의 신뢰성 점수 |
| `success_rate` | 실제 에이전트 호출 기반의 역대 성공률 |
| `pitfalls` | 주요 장애 유형 및 권장 완화 조치 |
| `alternatives` | 성능 기준으로 순위가 매겨진 최상의 대안 목록 |
| `jurisdiction` | GDPR 위험 및 데이터 레지던시 정보 |
| `latency` | 예상 응답 지연 시간 |

**3. 지능적 의사결정**

에이전트는 이를 바탕으로:

- 계획대로 해당 도구를 계속 사용하거나
- 더 나은 대안으로 자동 전환하거나
- 사용자에게 결정 사항을 제시할 수 있습니다

**4. 선택적 피드백 루프**

호출 후 에이전트는 익명 결과 보고서를 제출할 수 있습니다. 이 데이터는 강력한 **네트워크 효과**를 통해 모든 사용자의 점수를 지속적으로 개선합니다.

## 글로벌 에너지 절감 가능성

전 세계 모든 AI 에이전트와 챗봇이 ToolRate를 채택한다면, 에너지 측면에서 상당한 영향이 있을 것입니다.

1년 이내에 지구상의 활성 AI 에이전트 수가 인간의 수를 초과한다고 가정하고(>80억 에이전트), ToolRate가 실패하거나 낭비되는 도구 호출을 **60~75%** 줄일 수 있다면, 광범위한 도입은 매일 수십억 건의 불필요한 LLM 추론 및 재시도 루프를 방지할 수 있습니다.

> **보수적 추정:** ToolRate는 글로벌 AI 생태계에 연간 **8~15 TWh의 전력**을 절감시킬 수 있습니다 — 이는 **미국 일반 가정 150만~250만 가구**의 연간 소비량에 해당합니다.

절감 효과는 주로 다음에서 발생합니다:

- 실패한 API 호출 감소
- 토큰 낭비 절감
- 신뢰할 수 있는 도구로의 스마트 라우팅
