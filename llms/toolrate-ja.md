# ToolRate システム概要

## ToolRate とは？

ToolRate は、自律型 AI エージェント向けの**クラウドソーシングベースの信頼性レイヤー**です — エージェントが外部ツールや API を呼び出す*前に*、その信頼性を評価できるリアルタイムの信頼性オラクルです。

これはエージェント開発における最も重大な実践的問題の一つを解決します：ほとんどの障害は LLM 自体によって引き起こされるのではなく、外部ツールや API の予測不能な動作 — レート制限、スキーマドリフト、認証の問題、ボット対策、エッジケース — によって引き起こされます。

---

## ToolRate は誰のためのものですか？

- **本番環境対応**の AI エージェントを構築する開発者
- **LangChain、CrewAI、LangGraph、AutoGen** または **LlamaIndex** を使用するチームや個人開発者
- **GDPR とデータレジデンシー**を重視するヨーロッパの開発者
- デモでは正常に動作するものの、実際のシナリオで頻繁に失敗するエージェントに悩んでいるすべての方

---

## ToolRate の仕組み

システムは意図的にシンプルかつ軽量に設計されています：

**1. 呼び出し前チェック**

外部ツールや API を呼び出す前に、エージェントが ToolRate にクエリを送信します：

```python
assessment = toolrate.guard(tool_identifier=..., context=...)
```

**2. 構造化されたレスポンス**

ToolRate は以下の内容を含む JSON ペイロードを即座に返します：

| フィールド | 説明 |
|---|---|
| `reliability_score` | 0〜100 の信頼性スコア |
| `success_rate` | 実際のエージェント呼び出しに基づく過去の成功率 |
| `pitfalls` | よくある障害パターンと推奨される対策 |
| `alternatives` | パフォーマンス順にランク付けされた代替手段 |
| `jurisdiction` | GDPR リスクとデータレジデンシー情報 |
| `latency` | 推定レスポンスレイテンシ |

**3. インテリジェントな意思決定**

エージェントはその後：

- 予定通りにツールを使用して処理を続行する
- より優れた代替手段に自動的に切り替える
- 意思決定をユーザーに提示する

**4. オプションのフィードバックループ**

呼び出し後、エージェントは匿名の結果レポートを送信できます。このデータは強力な**ネットワーク効果**を通じて、すべてのユーザーのスコアを継続的に改善します。

## グローバルな省エネルギーの可能性

世界中のすべての AI エージェントとチャットボットが ToolRate を採用した場合、エネルギーへの影響は非常に大きくなります。

1年以内に地球上の活動中の AI エージェント数が人間の数を上回る（80 億以上のエージェント）と仮定し、ToolRate が失敗または無駄なツール呼び出しを **60〜75%** 削減できるとすれば、広範な普及により毎日数十億件の不要な LLM 推論と再試行ループを防ぐことができます。

> **保守的な試算：** ToolRate は世界の AI エコシステム全体で年間 **8〜15 TWh の電力**を節約できる可能性があります — これは**アメリカの一般的な家庭 150 万〜250 万世帯**の年間消費量に相当します。

節約は主に以下から生まれます：

- 失敗した API 呼び出しの削減
- トークンの無駄遣いの軽減
- 信頼性の高いツールへのスマートなルーティング

## 他のツールとの比較

| ツール | タイプ | 障害を予防？ | クラウドソーシングデータ | 代替手段を提供 | GDPR / 管轄 | 主な対象 |
|---|---|:---:|:---:|:---:|:---:|---|
| **ToolRate** | 呼び出し前信頼性オラクル | ✅ | ✅ | ✅ | ✅ 強力 | 本番環境エージェント |
| LangSmith | オブザーバビリティ + トレーシング | ❌ | ❌ | ❌ | ⚠️ 限定的 | LangChain エコシステム |
| Langfuse | オープンソース オブザーバビリティ | ❌ | ❌ | ❌ | ⚠️ 限定的 | オープンソース トレーシング |
| Braintrust | 評価 + トレーシング | ⚠️ 部分的 | ❌ | ❌ | ⚠️ 限定的 | 評価重視のチーム |
| Helicone | LLM + ツール オブザーバビリティ | ❌ | ❌ | ❌ | ⚠️ 限定的 | コストとレイテンシの監視 |
| AgentOps | エージェント モニタリング | ❌ | ❌ | ❌ | ⚠️ 限定的 | エージェント動作分析 |

> ToolRate は現在、実際のクラウドソーシングによるエージェント経験を活用して**予防的に機能する唯一のソリューション**です。
