# סקירת מערכת ToolRate

## מה זה ToolRate?

ToolRate הוא **שכבת אמינות מבוססת-קהל** לסוכני AI אוטונומיים — אורקל אמינות בזמן אמת שמאפשר לסוכנים להעריך עד כמה כלי חיצוני או API אמין *לפני* שהם קוראים לו.

הוא פותר אחת מהבעיות המעשיות הקריטיות ביותר בפיתוח סוכנים: רוב הכשלים אינם נגרמים על ידי ה-LLM עצמו, אלא על ידי התנהגות בלתי צפויה של כלים ו-API חיצוניים — הגבלות קצב, סחיפת סכמה, בעיות אימות, הגנות אנטי-בוט ומקרי קצה.

---

## למי מיועד ToolRate?

- מפתחים הבונים סוכני AI ברמת **סביבת ייצור**
- צוותים ומפתחים עצמאיים העובדים עם **LangChain, CrewAI, LangGraph, AutoGen** או **LlamaIndex**
- מפתחים אירופאים המקפידים על **GDPR ומיקום נתונים**
- כל מי שמתוסכל מסוכנים שעובדים היטב בהדגמות אך נכשלים לעתים קרובות בתרחישים מהעולם האמיתי

---

## כיצד עובד ToolRate

המערכת תוכננה בכוונה להיות פשוטה וקלת משקל:

**1. בדיקה לפני הקריאה**

לפני קריאה לכל כלי או API חיצוני, הסוכן מבצע שאילתה ל-ToolRate:

```python
assessment = toolrate.guard(tool_identifier=..., context=...)
```

**2. תגובה מובנית**

ToolRate מחזיר מיד מטען JSON המכיל:

| שדה | תיאור |
|---|---|
| `reliability_score` | ציון בין 0–100 |
| `success_rate` | שיעור היסטורי המבוסס על קריאות סוכן אמיתיות |
| `pitfalls` | מצבי כשל נפוצים + פתרונות מומלצים |
| `alternatives` | חלופות מובילות מדורגות לפי ביצועים |
| `jurisdiction` | סיכון GDPR ומידע על מיקום נתונים |
| `latency` | זמן תגובה משוער |

**3. החלטה חכמה**

הסוכן יכול אז:

- להמשיך עם הכלי כמתוכנן
- לעבור אוטומטית לחלופה טובה יותר
- להציג את ההחלטה למשתמש

**4. לולאת משוב אופציונלית**

לאחר הקריאה, הסוכן יכול להגיש דוח תוצאות אנונימי. נתונים אלה משפרים ברציפות את הציונים עבור כל המשתמשים באמצעות **אפקט רשת** חזק.

## פוטנציאל חיסכון גלובלי באנרגיה

אם כל סוכני ה-AI וה-Chatbot בעולם יאמצו את ToolRate, ההשפעה על האנרגיה תהיה משמעותית.

בהנחה שבתוך שנה אחת יהיו יותר סוכני AI פעילים מאשר בני אדם על פני כדור הארץ (>8 מיליארד סוכנים), וש-ToolRate יכול להפחית קריאות כלים כושלות או מבוזבזות ב-**60–75%**, אימוץ נרחב עשוי למנוע מיליארדי הסקות LLM מיותרות ולולאות ניסיון חוזר מדי יום.

> **הערכה שמרנית:** ToolRate יכול לחסוך לאקוסיסטם ה-AI העולמי בין **8 ל-15 TWh חשמל בשנה** — שווה ערך בקירוב לצריכה השנתית של **1.5 עד 2.5 מיליון משקי בית אמריקאים ממוצעים**.

החיסכון נובע בעיקר מ:

- פחות קריאות API כושלות
- הפחתת בזבוז טוקנים
- ניתוב חכם יותר לכלים אמינים
